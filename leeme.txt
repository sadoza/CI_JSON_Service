En este proyecto tenéis un ejemplo de como se puede utilizar el nucleo del 
Framework CodeIgniter compartido por más de una aplicación. Pare conseguir 
esto se ha realizado las siguientes operaciones:

- Se ha copiado la carpeta "ci_system" en otra ruta. Esta en un sistema de 
producción podría estar ubicada en una carpeta que no sirva el servidor Web

- Se ha modificado el fichero "index.php" para que la variable $system_path
apunte a la nueva ruta en la que se encuentra el nucleo del Framework
        $system_path = '../ci_system';

De esta forma tan solo tenemos que copiar las veces que se precisen la carpeta 
de nuestra aplicación y solo tendremos una copia del Framework.
